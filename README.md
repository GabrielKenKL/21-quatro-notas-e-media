Programa em Python que cadastra alunos, solicita quatro notas, calcula a média aritmética e salva os dados em um arquivo de texto no diretório /tmp/exercicios. O diretório é criado automaticamente com permissão 744, e o arquivo armazena cada aluno em uma linha no formato nome;nota1;nota2;nota3;nota4;media.
